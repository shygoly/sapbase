# WebSocket Real-time Collaboration

## Installation

Add dependencies to `package.json`:

```bash
npm install @nestjs/websockets socket.io
npm install --save-dev @types/socket.io
```

## Quick Start

1. The WebSocket module is already integrated into `AppModule`
2. Start the server - WebSocket will be available on the same port
3. Connect from frontend using the WebSocket client

## Features

- ✅ Real-time notifications
- ✅ Collaborative editing
- ✅ Presence awareness
- ✅ JWT authentication
- ✅ Room management
- ✅ Automatic reconnection
